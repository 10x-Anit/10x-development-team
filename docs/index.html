3 Ways to Use It

Pick the one that fits your workflow

MCP Server (Any AI Client)

npx @10x-dev/mcp-server setup

Auto-detects Claude Desktop, Claude Code, Codex, OpenCode. One command configures everything.

Claude Code Plugin (Direct)

npx @10x-dev/mcp-server install-plugin .

Copies the plugin into your project. Use slash commands: /10x-development-team:start

npm Package

npm install -g @10x-dev/mcp-server

Install globally. Run 10x-mcp anywhere. Use doctor to verify.

What Makes It Different

Works Everywhere

MCP server works with Claude Desktop, Claude Code, Codex, OpenCode, and any MCP-compatible AI client. One plugin, any tool.

Connect Any Data

Supabase, Firebase, Airtable, Google Sheets, Notion, REST APIs, CSV files. Your data, your app.

Persistent Memory

SQLite database remembers every project across sessions. Switch between 20 apps, resume any of them with full context.

40 Knowledge Files

Agents copy proven code patterns — not reinvent. Next.js, Tailwind, Prisma, Stripe, Resend, and more.

Model-Aware

Small models copy code exactly. Large models enhance with animations, accessibility, dark mode, and SEO.

Token-Efficient

Project index means agents read one file instead of scanning hundreds. Saves tokens, saves money.
